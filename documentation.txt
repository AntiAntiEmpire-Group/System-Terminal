System Terminal
Версия 1.0.0


Описание:
    Терминал, позволяющий вводить команды с дальнейшем их выполнении. 


Возможности:
    Поддержка подсказок при допущении написании команды;
    Поддержка исключений;


Команды:
    echo <string> - вывод <string> в терминал;

    clear - очистка терминала;

    help - вывод всех команд ;
    
    color <flag> - изменения цветовой темы в терминале
        flag: 
            -i - информация о текущей цветовой темы;
            0 - стандартная цветовая тема;
            1 - серо-зелёная;
            2 - красная тема;
